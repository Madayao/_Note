# JavaScript 语句总结

| 语法                            | 名称         | 解释                                       | 例                          |
| ----------------------------- | ---------- | ---------------------------------------- | -------------------------- |
| `for (variable in object) {}` | `for...in` | 任意顺序遍历对象可枚举属性                            | `for(let key in object){}` |
| `for (variable of object) {}` | `for...of` | 迭代可迭代对象属性值, object: Array, String, arguments, Map, Set, TypedArray | `for(let char in array){}` |
