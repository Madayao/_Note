# Buffer

| 代码                            | 解释                      | 实例   |
| ----------------------------- | ----------------------- | ---- |
| `Buffer.from(string)`         | 将 string 数据转成 buffer 类型 |      |
| `Buffer.concat([buf1, buf2])` | 将两个 buffer 拼接在一起        |      |
| `buffer.byteLength`           | buffer 类型数据的长度          |      |
